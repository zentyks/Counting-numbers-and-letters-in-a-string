Подсчёт букв и цифр
📌 Описание

Простая программа, которая считает количество букв и цифр в введённой строке.

🚀 Особенности
проверяет каждый символ строки
считает отдельно буквы
считает отдельно цифры
игнорирует другие символы (пробелы, знаки)

✍️ Примеры

Ввод:
hello123

Вывод:

Количество букв: 5
Количество цифр: 3
